Grid system

Basis has 2 - 12 based grid system. For example .bs-item--1-2, .bs-item--12-12. When using responsive Grids, Adding the specific breakpoints at class names. For example .bs-item--md-1-2, .bs-item--md-12-12.

Key Applies Class name
None Always .bs-item--1-12
sm >= 768px .bs-item--sm-1-12
md >= 992px .bs-item--md-1-12
lg >= 1200px .bs-item--lg-1-12
