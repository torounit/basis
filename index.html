Grid system

Basis has 2 - 12 based grid system. For example ._col--1-2, ._col--12-12. When using responsive Grids, Adding the specific breakpoints at class names. For example ._col--md-1-2, ._col--md-12-12.

Key Applies Class name
None Always ._col--1-12
md >= 640px ._col--md-1-12
lg >= 1024px ._col--lg-1-12
